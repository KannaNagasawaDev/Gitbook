# Вступ

Kawa Fedora Gitbook - це проста книга зроблена для того, щоб я змогла поділитися своїм досвідченням, й допомогти новачкам пізнати цей світ Linux, також ця книга може бути допоміжним інструментом і для досвітчаних користувачів.&#x20;

У цій книзі я намагаюся пояснювати речі простою мовою, щоб навіть ті, хто лише починає знайомство з Linux, могли легко розібратися та зробити свої перші кроки.

Цей GitBook також може бути корисним і для більш досвідчених користувачів як допоміжний довідник або джерело практичних порад.

Моя мета — зробити шлях у світ Linux трохи простішим і зрозумілішим для кожного.

{% hint style="info" icon="triangle-exclamation" %}
Цей GitBook не є офіційною документацією Fedora Linux, тому всі дії, описані тут, ви виконуєте на власний страх і ризик.
{% endhint %}
